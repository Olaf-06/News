Первая версия первого ios проекта, в котором содержится функциональный минимум. Представляет из себя новостную ленту, взятую из сайта Newsapi.org. Планируется добавить поиск, возможность расскрыть новость на новом экране и обновить дизайн под LiquidGlass. Также планируется добавить слои для разграничения представления и работы с данными, подстроить приложение под современный стек.
